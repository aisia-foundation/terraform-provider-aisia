# Security Policy

## Reporting a vulnerability
Email **security@aisia.fr** (ou admin@aisia.fr). Merci de ne pas ouvrir d'issue
publique pour une faille. Réponse sous 72h.

## Signature des releases
Les binaires du provider sont signés GPG (clé `96D2AD35E6565778`). Vérifiez la
signature `*_SHA256SUMS.sig` avant usage hors registry.
